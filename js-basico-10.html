Operadores que vamos a ocupar para hacer ciertas cosas

  • Operadores aritmeticos o binarios: nos van a ayudar a hacer operaciones matematicas, se le llama binario porque hace operaciones con dos valores y genera un resultado
  • Operadores logicos o unitarios: tienen un solo operador trbajando con un solo valor
  • Operadores de asignacion: asigna valores para inicializar variables
  • Operadores de comparación: comparar valores
