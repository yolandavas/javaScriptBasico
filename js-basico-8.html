¿Qué es Coerción?

Coerción es la forma en la que podemos cambiar un tipo de valor a otro (Por ejemplo un tipo numero lo podemos cambiar a un tipo string), existen dos tipos de coerción:
  • Coerción implicita: es cuando el lenguaje nos ayuda a cambiar el tipo de valor.
  • Coerción explicita: es cuando obligamos a que cambie el tipo de valor.

4 + "7"; //28 string
4 * "7" // 28 numero
2 + true; // 3
false - 3 // 3

Ejemplo de cohercion implicita

var a = 4 + "7"; //funcion de un valor a otro valor
typeof a
"String" //lo convierte en string
El operador de + en variables o vairables de tipos diferentes genera la concatenación

Ejemplo de cohercion explicita

Var x=10
var y = x + " "
typeof x
"number"
var z = String(x); //cohercion explicita
typeof x
"string"
var a = Number(z)
typeof a
"number"
